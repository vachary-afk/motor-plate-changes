# Motor Plate Changes

Interactive mobile prototype for the Motor 2.0 "share plate details for instant quotes" flow.

Guides the user through three steps — **Emirate**, **plate code**, and **plate number** — with:

- An emirate-aware plate-code keyboard (Dubai uses letters A–Z; all other emirates use a numeric code)
- A live number-plate preview that updates as you type
- Inline validation and error states
- A gated tab flow with a slide-in Continue button

Open [`index.html`](index.html) in a browser to run it. It's a single self-contained file — no build step or dependencies.
